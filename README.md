# Awesome Avatars

> A curated collection of tools, services, standards, and communities for digital avatars - from 2D profile pictures to AI-powered 3D avatars with voice and motion capture.

**Created by Lia & Yeo | November 2025**

## 1. 2D Avatar Generators

### 1.1 Static Generators
- [**Gravatar**](https://gravatar.com/) - Globally Recognized Avatars, email-based
- [**DiceBear**](https://dicebear.com/) - Open source avatar library with multiple styles
- [**Robohash**](https://robohash.org/) - Generates unique robot/monster avatars from any text
- [**UI Faces**](https://uifaces.co/) - Curated avatars for UI mockups
- [**Adorable Avatars**](http://avatars.adorable.io/) - Cute, generated avatar faces
- [**Boring Avatars**](https://boringavatars.com/) - Minimalist, geometric avatars

### 1.2 Customizable Builders
- [**Avataaars**](https://getavataaars.com/) - Sketch-based avatar library by Pablo Stanley
- [**Bitmoji**](https://www.bitmoji.com/) - Personalized cartoon avatars for messaging
- [**Picrew**](https://picrew.me/) - Japanese avatar maker with various artist styles
- [**Artbreeder**](https://www.artbreeder.com/) - Generate portraits by mixing different faces

## 8. Avatar Standards & Formats

- [**VRM**](https://vrm.dev/en/) - Open standard for humanoid 3D avatars (VR-optimized)
- [**GLB / GLTF**](https://www.khronos.org/gltf/) - Standard 3D format, web-compatible
- **FBX** - Autodesk format for animation and rigging
- **OBJ** - Universal 3D mesh format
- [**USD / USDZ**](https://developer.apple.com/augmented-reality/quick-look/) - Universal Scene Description (Apple AR)
- **Collada (DAE)** - Open standard for 3D asset exchange
- [**Live2D Cubism Format**](https://www.live2d.com/en/) - Proprietary for 2D character animation
- **PMX / MMD** - MikuMikuDance format (popular in Japan)

## 9. Face/Body Capture Hardware

### 9.1 Face Tracking
- **iPhone with ARKit** - Face ID-based tracking
- **Android ARCore** - Google's AR framework for face tracking
- **Webcams** - Budget solution with software-based tracking
- [**Faceware**](https://facewaretech.com/) - Professional facial capture system
- **Dynamixyz** - Marker-less face tracking

### 9.2 Full Body Motion Capture
- [**Xsens**](https://www.movella.com/products/motion-capture) - Inertial motion capture suits (professional)
- [**OptiTrack**](https://optitrack.com/) - Optical motion capture system
- [**Vicon**](https://www.vicon.com/) - High-end optical mocap
- [**Rokoko**](https://www.rokoko.com/) - Affordable Smartsuit Pro
- [**Perception Neuron**](https://neuronmocap.com/) - Budget-friendly inertial mocap alternative

### 9.3 Budget Solutions
- **Kinect** (Xbox) - Budget depth camera for body tracking
- [**RealSense**](https://www.intel.com/content/www/us/en/architecture-and-technology/realsense-overview.html) (Intel) - Depth cameras for tracking
- **PS Move** - PlayStation controller as budget tracking
- **Webcam + AI** - Software-only tracking (OpenPose, MediaPipe)

### 9.4 DIY
- [**MediaPipe**](https://mediapipe.dev/) - Google's ML framework for pose/face/hand tracking

- [**DeepMotion**](https://www.deepmotion.com/) - AI-based mocap from videos

- [**VSeeFace**](https://www.vseeface.icu/) - Uses webcam for 3D avatar tracking (free)
